# Web_projesi
MediStore - Medikal ürün satış ve tanıtım platformu
## Proje açıklaması
Bu projenin amacı, medikal ürünlerin internet ortamında tanıtımını yapmak ve kullanıcıların bu ürünleri online olarak inceleyip satın alabilmesini sağlamaktır. Sistem, kullanıcı dostu arayüzü sayesinde sağlık ürünlerine hızlı ve güvenli erişim sunmayı hedeflemektedir
## Hedef kullanıcılar
Hasta
Eczaneciler
Medikal şirketi
## Temel özellikleri
Ürün görseli
Ürün adı
Kısa açıklama 
Fiyat bilgisi
Sepete ekle butonu
## Kullanılan teknolojiler
HTML
CSS
Javascript
PHP
MySQL
